C언어 개요

C언어

C언어는 현재 사용되고 있는 거의 모든 컴퓨터 시스템에서 사용할 수 있는 프로그래밍 언어입니다.

현재 널리 사용되는 주요 운영체제의 커널은 대부분 C언어를 이용해 구현되어 있습니다.

이처럼 C언어는 시스템 프로그래밍에 가장 잘 어울리지만, 응용 프로그래밍에도 많이 사용되는 프로그래밍 언어입니다.

 

C언어 수업 시작 =>

예제

#include <stdio.h>

#define TEXT "Welcome to C Programming!!"  

 

int main()

{

    printf(TEXT);

    return 0;

}
