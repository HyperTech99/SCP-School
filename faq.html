FAQ
[[ FAQ - 자주 묻는 질문 ]]
  • Q. 어떤 버전의 Visual Studio를 설치해야 하나요?
       A. 기본적으로 IDE는 개인이 사용하기 편한 것을 사용하면 됩니다. 다만, Visual Studio 2017이나 Visual Studio 2013, Visual Studio 2010 중에 1개를 사용하기를 권장합니다. 1학년 C 프로그래밍 강의에서는 Visual Studio 2010을 사용합니다.
  • Q. 분명히 올바르게 코딩하였는데 "error C4996: 'scanf': This Function or variable may be unsafe. Consider using scanf_s instead. To disable deprecation, use _CRT_SECURE_NO_WARNINGS. See online help for details" 오류는 왜 나죠?
       A. #define _CRT_SECURE_NO_WARNINGS 를 코드 최상단에 입력하거나, 혹은 프로젝트 속성 > C/C++ > 전처리기 메뉴에 들어가서 전처리기 옵션 란에 _CRT_SECURE_NO_WARNINGS를 추가하세요. scanf 함수가 안전하지 않기 때문입니다. OJ 채점 시스템에서는 scanf_s 함수는 지원하지 않습니다.
  • Q. SCP School에는 C 언어 외의 다른 언어의 팁이나 해설은 없나요?
       A. 추후 JAVA나 Python, C++ 관련 컨텐츠 추가를 검토하고 있습니다.
